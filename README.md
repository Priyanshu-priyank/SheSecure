# SheSecure
SheSecure is a smart women safety application designed to ensure real-time protection and safe navigation. It empowers users to send instant SOS alerts, share live location with trusted contacts and nearby users, and navigate through AI-recommended safe routes based on real-world data and community feedback.

Features:
🆘 One-Tap SOS Alert — Instantly notifies nearby users, police, and selected emergency contacts via push notification and email.

📍 Live Location Tracking — Detects and shares user’s real-time location during emergencies.

🗺️ Smart Map View — Displays safe and unsafe zones using community reviews and past crime data.

🛣️ Safe Route Recommendation — Suggests safer navigation paths using AI-based risk estimation.

⭐ Area Rating System — Users can rate the safety of areas they visit.

🔔 Proximity Alerts — Sends notification when entering a high-risk zone.

👩‍💻 Police Dashboard — Real-time SOS monitoring interface for law enforcement or volunteers.
